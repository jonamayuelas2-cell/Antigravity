# Antigravity

PWA portfolio para acceder rapidamente a los proyectos creados con Antigravity.

## Proyectos enlazados

- Roma Eterna: https://github.com/jonamayuelas2-cell/Roma2
- Restaurante Mexicano: https://github.com/jonamayuelas2-cell/RestauranteMexicano

## Ejecutar localmente

```bash
python -m http.server 8081
```

Abrir:

```text
http://127.0.0.1:8081/
```

## Archivos

- `index.html`: estructura principal
- `style.css`: diseno responsive
- `app.js`: interaccion de tarjetas y registro del service worker
- `manifest.json`: configuracion PWA
- `sw.js`: cache basica
- `icons/icon.svg`: icono instalable
